# espanso-radiology

Personal Espanso configuration and match files for radiology reporting workflow in brazilian portuguese.


## Setup

1. Download the [Espanso portable release](https://github.com/espanso/espanso/releases/latest) for your OS
2. Clone this repo and copy `config/` and `match/` into your Espanso config directory (`espanso path`)
3. Start Espanso
